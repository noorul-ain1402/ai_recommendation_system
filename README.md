# 🎬 AI Movie Recommendation System

A simple Python-based recommendation system that suggests Bollywood movies based on the user's preferred genre and favorite actor.

FEATURES

- Genre-based movie recommendations
- Favorite actor matching
- Match score calculation
- Recommendation history
- Menu-driven interface

TECHNOLOGIES USED

- Python
- Dictionaries
- Lists
- Loops
- Conditional Statements

HOW TO RUN

```bash
python bollywood_recommendation_system.py
```

 AVAILABLE GENRES
- Action
- Romance
- Comedy
- Drama
- Thriller
